# User Management API

This project is a simple ASP.NET Core Web API for managing users.

## Features
- CRUD operations for users
- GET, POST, PUT, and DELETE endpoints
- Input validation
- Custom logging middleware
- Swagger/OpenAPI support

## Technologies
- ASP.NET Core
- .NET 8
- C#
- Swagger

## Copilot Usage
Microsoft Copilot was used to help generate the initial API structure, debug routing and validation issues, and improve the custom logging middleware.

## How to Run

```bash
dotnet run
```

## Then open Swagger:
https://localhost:xxxx/swagger
